qtau
====

Free opensource UTAU-like editor for singing synthesizers, uses Qt.
* License: [WTFPL] [1]
* Dependencies: Qt 4.8
* OS: any supported by Qt

Linked projects: [lesynth] [2], [lauloid] [3], [Rocaloid] [4]

Discussion: issues here, [twitter] [5] or #vocadb at irc.rizon.net
